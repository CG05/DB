soup.select(*)
- 모든 요소 선택

soup.select(div)
- 요소 이름을 선택

soup.select(.main)
- 클래스 이름을 선택

soup.select(#main)
- 아이디 이름을 선택

soup.select(div, a)
- 여러 요소를 선택

soup.select(div ul li a)
soup.select(div > ul > li > a)
- 앞선택자의 후손 중 요소 선택

soup.select(h3 + div)
- 같은 계층에서 뒤에 있는 요소 선택

soup.select(h3 - div)
- 앞 요소 부터 뒤 요소 까지 요소를 모두 선택

soup.select(a[href])
- 해당 속성을 가진 요소를 선택

soup.select(div[class=main])
- 속성의 값이 지정한 값과 같은 요소를 선택

soup.select(div[class~=main])
- 속성의 값이 지정한 값과 똑같은 요소만 선택

soup.select(div[class|=main])
- 속성의 값으로 시작하는 요소 선택

soup.select(div[class^=main])
- 속성의 값으로 시작하는 요소 선택

soup.select(div[class$=main])
- 속성의 값으로 끝나는 요소 선택

soup.select(div[class+=main])
- 속성의 값이 포함되어 있는 요소 선택
